<h2>💡Travel Information Web Using JPA </br></h2>

<h1>프로젝트 소개</h1>
 - 웹을 통해 여행 정보에 대한 커뮤니티<br/>
 - 각 나라의 지역에 따른 여행 경보 제공<br/>
 - 나라 도시의 현재 날씨 제공<br/>
 - 구글 지도 제공 

 <h1>사용 기술</h1>
 - Spring Boot<br/>
 - JPA<br/>
 - JavaScript<br/>
 - HTML<br/>
 - CSS

 <h1>사용 오픈 API</h1>
 - Google Map<br/>
 - 공공 API (외교부 국가 지역별 여행경보)<br/>
 - OpenWeatherMap<br/>
